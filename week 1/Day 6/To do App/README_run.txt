Run:
python to_do_list_app.py

Menu options:
1 Add task (prevents duplicate names, case-insensitive)
2 View tasks
3 Mark completed by number
4 Delete task by number
5 Edit task (rename) by number (prevents duplicates)
6 Search tasks by keyword
7 Pending vs Completed counts (Total/Pending/Completed)
8 Sort tasks (Pending first)
9 Exit
