# 🧠 Key Concepts Learned

## JSON

* JSON is a language-independent format for storing and exchanging structured data.
* Convert Python dictionaries/lists to JSON using `json.dump()`.
* Read JSON files using `json.load()`.
* Use `indent=4` for readable JSON formatting.

### Functions Learned

* `json.dump()`
* `json.load()`
* `json.dumps()`
* `json.loads()`

---

## CSV

* CSV (Comma-Separated Values) stores tabular data.
* Create CSV files using `csv.writer()`.
* Read CSV files using `csv.reader()`.
* Write dictionaries using `csv.DictWriter`.
* Read dictionaries using `csv.DictReader`.
* Use `newline=""` when writing CSV files on Windows.

---

## JSON vs CSV

| JSON                            | CSV                               |
| ------------------------------- | --------------------------------- |
| Stores structured/nested data   | Stores tabular data               |
| Used in APIs                    | Used in spreadsheets and datasets |
| Supports lists and dictionaries | Stores rows and columns only      |
